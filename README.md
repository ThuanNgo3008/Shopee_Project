# Shopee Interface Clone - Frontend Practice

This project simulates the interface of Shopee's product category page, focusing on developing skills in building realistic user interfaces (UI) and optimizing the user experience.

## 🔗 Live Demo
https://thuanngo3008.github.io/Shopee_Project/src/
## 🛠️ Technology used
* **Frontend:** HTML5, CSS3, JavaScript/jQuery.
* **Responsive:** Displays well on all devices (Mobile, Tablet, Desktop).
* **HTML5:** Building a website structure that is SEO-friendly.
* **CSS3:** Using Grid, Flexbox, and animation effects.
* **Bootstrap:** Supports fast layout creation and good responsiveness on mobile devices.
* **Deployment:** GitHub Pages.
## ✨ Features
* Grid & Flexbox Layout: Organizes product lists, filters, and navigation bars scientifically and precisely according to the original design.
* Hover Effect: Product cards have a prominent effect when users interact with them.
* Component Structure: Clearly separates the header, sidebar (categories), and main product display area.
